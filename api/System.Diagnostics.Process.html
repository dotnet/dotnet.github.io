Note that a new process that’s started alongside already running instances of the same process will be independent from the others. In addition, Start may return a non-null Process with its HasExited property already set to true. In this case, the started process may have activated an existing instance of itself and then exited.

WaitForExit(Int32)

Instructs the Process component to wait the specified number of milliseconds for the associated process to exit.

Declaration
public bool WaitForExit(int milliseconds)
Parameters
Type Name Description
Int32 milliseconds

The amount of time, in milliseconds, to wait for the associated process to exit. The maximum is the largest possible value of a 32-bit integer, which represents infinity to the operating system.
Returns
Type Description
Boolean

true if the associated process has exited; otherwise, false.
